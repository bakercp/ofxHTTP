Detailed Description

A simple default client redirect policy.

This redirect policy will attempt to redirect response codes: 

- 301 Moved Permanently
- 302 Found
- 303 See Other
- 307 Temporary Redirect

Additionally, this policy will will redirect content containing requests such as POST and PUT by converting their request type to a GET.
