const Poco::UUID& ofx::HTTP::BasePostRequestResponseEventArgs::getPostId ( ) const
inline

The post id.

Each time a POST request, is processed it is assigned a unique id. This id allows us to track post progress updates and multi-part posts.

Returns
the session id or Poco::UUID::null if not set.
