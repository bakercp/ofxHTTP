Detailed Description

A JSON Web Token (JWT) generator.

JSON Web Tokens are submitted to authentication servers in order to acquire an authentication token for a given API. The authentication token is then submitted with subsequent API requests.

JWTs have the following format:

aaaaaaaaaa.bbbbbbbbbbb.ccccccccccc
