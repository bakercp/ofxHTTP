std::streamsize ofx::HTTP::ClientResponseProgressArgs::getContentLength ( ) const
virtual

Get the total content length associated with this request.

If the total content length is unknown (e.g. during chunked-transfer) Poco::Net::HTTPMessage::UNKNOWN_CONTENT_LENGTH is returned.

Returns
the total length of the content in bytes iff the content length is known. Otherwise returns Poco::Net::HTTPMessage::UNKNOWN_CONTENT_LENGTH.
