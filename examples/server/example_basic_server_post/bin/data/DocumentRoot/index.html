ofx::HTTP::SimplePostServer Demo

Simple Post Server Demo


ofx::HTTP::SimplePostServer is a simple server implements an HTML form server.


Example Form Using text/plain Form Encoding

The following form is defined by <form role="form" action="/post" method="post" enctype="text/plain">. This for will send an event to the bool onHTTPPostEvent(ofx::HTTP::PostEventArgs& evt); callback. In this case, the content cannot be reliably parsed so it is up to the user to extract name / value pairs.

Example Form Using application/x-www-form-urlencoded Form Encoding

The following form is defined by <class="form-inline" role="form" action="/post" method="post" enctype="application/x-www-form-urlencoded">. This will send en event to the bool onHTTPFormEvent(ofx::HTTP::PostFormEventArgs& evt); callback. In this case, the content will be parsed and returns as a set of name / value pairs.

Example Form Using multipart/form-data Form Encoding

The following form is defined by <role="form" action="/post" method="post" enctype="multipart/form-data">. This will send events to three callbacks. If files are attached, each file will be processed in sequence and will send events at the beginning of the upload, during progress (as determined by the size of the upload buffer) and one upon completion. The callbacks are as follows:
  • bool onHTTPPostEvent(ofx::HTTP::PostEventArgs& evt);
  • bool onHTTPFormEvent(ofx::HTTP::PostFormEventArgs& evt);
  • bool onHTTPUploadEvent(ofx::HTTP::PostUploadEventArgs& evt);
