Detailed Description

Represents an abstract WebSocketExtension factory.

Can process and respond to headers in WebSocket Upgrade request. Returns an approprately configured AbstractWebSocketFilter for use with a WebSocketConnection.

Create an AbstractWebSocketFilter based on the ServerEventArgs.

This function will set the appropriate response headers in the evt.getResponse() if applicable.

The function will return an AbstractWebSocketFilter or a nullptr if no matching AbstractWebSocketFilter can be configured.
